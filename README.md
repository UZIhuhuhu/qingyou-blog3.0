## 青柚工作室主页

### GreenPomelo-homepage

青柚工作室成立于 2017 年 10 月，现已有超过 30 位成员分别负责技术、设计、运营推广等工作。截止到现在，青柚工作室已经开发“南京邮电大学"、“校谈”等微信小程序。其中，“南邮小程序”现已有超过 3.9 万用户，日均活跃量超过 8 千人次。
